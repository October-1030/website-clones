# Product Suite Specification

## Overview
- Target: `src/components/AskSiaLanding.tsx` (`ProductSuite`)
- Interaction: click-to-select tabs plus smooth scroll to one of five panels.
- Reference: background `rgb(248,250,255)`, section padding 100px, sticky tab rail.

## Products
网页应用, 移动端, 扩展, 资料库, AskSia 智能体. Every panel has an eyebrow, title, paragraph, three bullets, CTA and a layered UI mockup.

## Responsive
Desktop uses two columns (copy + visual). Mobile stacks the copy and visual, while the tab rail scrolls horizontally.
