# Header Specification

## Overview
- Target: `src/components/AskSiaLanding.tsx` (`Header`)
- Interaction: click-driven dropdowns and mobile menu; fixed on scroll.
- Exact reference: 80px height, white background, subtle `0 1px 24px rgba(0,0,0,.04)` shadow.

## Content
AskSia mark; 学生专区; 实用工具; 资源; 价格; ZH-CN; 登录; 下载 App 🔥. Mobile replaces the nav/actions with `Menu`.

## Responsive
Desktop header inner max-width 1440px with 48px side padding. At 900px and below, only mark and Menu button remain. Mobile menu is a rounded white overlay under the fixed header.
