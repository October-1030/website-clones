# AskSia Clone

This folder contains the standalone AskSia website clone, including its source code, downloaded assets, visual references, research notes, and extraction scripts.

## Run locally

```bash
npm install
npm run dev
```

Useful checks:

```bash
npm run lint
npm run typecheck
npm run build
```

The original target-specific material is grouped under:

- `docs/` — PRD, screenshots, behavior notes, topology, and component specs
- `public/images/asksia/` — downloaded AskSia assets
- `scripts/` — AskSia extraction and QA scripts
