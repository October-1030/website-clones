常见问题

还有些疑问？

这里是我们被问最多的几个问题 — 没看到答案？加客服群直接问

01Storybound 跟剪映、度加剪辑有什么不同？
剪映 / 度加剪辑是剪辑工具，需要你自己写脚本、找素材、配音。
Storybound 是"文案 → 剪映草稿"的全自动管道 — 我们帮你跑完 AI 改写、配图、配音的所有环节，最后产出剪映可直接打开的工程文件。 你只需要在剪映里做最后微调。
02下载就能用吗？要付费才能开始吗？
不用。下载安装后默认送 5 次完整流程试用机会。
你只需要自己接入 LLM、生图、TTS 的 API Key / sessionID（火山引擎、即梦、DeepSeek、通义、Kimi、MiniMax 等主流服务，也支持 OpenAI / Claude 兼容协议自定义接入），就能跑通完整流程。
觉得好用了再考虑买月卡 / 年卡 / 永久卡 — 软件能力一开始就完整给到，付费是为了解锁更多次数、模板、客服群、返佣等增值权益。
03一定要会用剪映吗？
会基础操作就行（拖时间轴、调字幕大小、改 BGM 音量这种）。Storybound 出来的草稿已经把字幕、音频、图片、BGM 全部对齐好了， 大部分时候你只需要在剪映里看一遍、调几个字幕、点导出就完事。
04生成的视频版权归谁？
归你。Storybound 只是工具，所有产出（文案、图片、音频、剪映草稿）的版权和使用权完全归你所有， 包括商用。但请确保你输入的原始文案本身合规、不侵权。
05需要联网吗？数据安全吗？
软件是本地桌面应用，但调用 AI 服务（LLM / 绘图 / TTS）需要联网。
你的文案、生成的图片、音频只保存在你本地，我们不上传任何创作内容到服务器。 API Key / sessionID 安全存储在系统钥匙串（Windows Credential Manager / macOS Keychain）。
06支持 Mac 吗？
✅ 同时支持 Windows 10/11 和 macOS 12+，Apple Silicon 原生构建。
07出图失败怎么办？
内置 5 级自动降级：(1) 单图失败自动重试；(2) 风控拦截自动改写提示词重新生成； (3) LLM 端二次改写规避敏感词；(4) 切换备用生图引擎；(5) 仍然失败的会用兜底图占位，你可以在画廊里单张重生。 —— 不会卡死整条任务。
08怎么找客服？
主页右下角有客服微信浮动按钮，扫码加群。月卡 / 年卡用户进 Storybound 软件服务群，永久卡用户进独立 VIP 群 + 陪跑 + 一对一答疑。
09退款政策？
7 天无理由全额退款。如果你试用 7 天内觉得不合适，发邮件给我们就退。 积分套餐已使用的部分按消耗扣除，剩余按比例退。
