# 产品模式与运行流程

## 1. 创建首页

创建页提供三个主入口：

| 入口 | 定位 | 典型产物 |
| --- | --- | --- |
| 图文任务 | AI 分镜配图 + 文案朗读 + 字幕 | 口播、人物故事、健康图书、带货视频的剪映草稿 |
| HTML 动画视频 | 代码驱动的知识卡片、金句排版和动态强调 | 可预览、可继续编辑的动画视频和 MP4 |
| 音乐 MV | 写词作曲或导入歌曲，按歌词生成/匹配画面 | 歌词卡点 MV、动态分镜和剪映草稿 |

创建页还显示最近任务，可按任务类型和状态继续编辑。

## 2. 图文任务

### 2.1 三种执行模式

| 模式 | 代码值 | 行为 |
| --- | --- | --- |
| 全自动 | `auto` | 完整执行全部 7 步 |
| 半自动 | `semi_auto` | 跳过 Step 0 预审与 Step 1 改写，从已有正文开始 |
| 直接出片 | `direct` | 进一步跳过智能分句，按空行机械切分后继续 |

三种模式不是三个不同页面，而是同一条流水线的不同起点。

### 2.2 七步流水线

| 步骤 | 名称 | 主要产物 |
| --- | --- | --- |
| 0 | 文案预审 | 清理广告、风险词和审阅备注 |
| 1 | 智能改写与封面生成 | 改写正文、封面标题、发布文案、标签、评论 |
| 2 | 影视分镜分句 | 可配图的句子/场景单元 |
| 3 | 生成绘图提示词 | 每个分镜的画面提示词或素材检索词 |
| 4 | 批量生图 | 分镜图片，可局部重画、裁剪、替换和批量导入 |
| 5 | TTS 配音 | 分句音频，或播客模式下的整篇双人音频 |
| 6 | 生成剪映草稿 | 草稿 JSON、媒体资源和可直接打开的剪映工程 |

### 2.3 交互式运行能力

- 可在指定步骤后暂停，人工确认后继续。
- Step 1/2/3 的产物可直接编辑，保存后从下一步续跑。
- 任一步可以“从这里重跑”，前置产物继续复用。
- 对缺图、缺音频等失败项可只修复失败条目，不重做成功项。
- 支持取消并保留断点、重启后继续、任务排队和批量顺序执行。
- 任务详情包含产物预览、配音试听、分镜画廊和发布素材。
- 修改图片、字幕、音色或模板后，可只重新打包草稿。

### 2.4 内容与素材分支

- 内置赛道至少包括人物故事、健康图书、电商带货；模板体系还覆盖民间故事等细分方向。
- 素材来源包括 AI 绘图、免版税网络素材、人物素材库真图。
- 文案可以来自粘贴原文、对标文案、选书商品资料、网络检索或 IMA 知识库。
- 配音可走火山引擎、MiniMax，或让用户上传外部生成/录制的音频。
- `podcast` 是图文任务里的脚本/配音分支，不是单独主入口：双人对话整篇合成，不能逐句重生；可使用单张封面填满视频。

## 3. HTML 动画视频

### 3.1 输入模式

- AI 命题创作：只给主题和目标字数，从零生成口播稿。
- 粘贴已有文案：可润色并切场景，也可保持原文只做分镜规划。

### 3.2 流程

1. 改写 + 分句。
2. 场景规划：为每个场景选择版式、标题、字幕和画面提示词。
3. 素材生成：背景图和透明前景 PNG。
4. TTS 配音。
5. 在 WebView 中真实渲染并预览每个动画场景。
6. 逐帧截图，经 FFmpeg 合成为 MP4。

当前包里识别到至少 16 种、实际 17 个布局 ID，包括满屏金句、全屏大图、左字右物、右字左物、四格盘点、左右对比、数据强调、人物聚焦等。字幕还提供打字机、卡拉 OK、逐字块、弹入、横扫、显影、呼吸等动效。

该模式支持单场景重配音、重生素材、手改字幕、用本地 ASR 重新对齐、重新套用草稿模板而不重跑图像和配音。

## 4. 音乐 MV

### 4.1 歌曲来源

- AI 写词作曲：按主题和风格生成歌词，并调用 MiniMax 作曲。
- 本地音乐：导入歌曲和歌词，跳过作曲。
- 翻唱新作：分析参考歌曲并按新主题改词，保持原曲结构和节奏；界面明确提示用户自行确认版权风险。
- 视频混剪入口已存在，但当前 UI 标记为开发中。

### 4.2 主要阶段

1. 歌曲/歌词准备与结构分析。
2. 歌词确认，可在作曲前暂停。
3. 按语义把歌词合并成画面分组。
4. 分句确认。
5. 为每组生成绘图提示词。
6. 提示词确认。
7. 批量生图或匹配上传的视频素材。
8. 图片确认，可批量重画、替换、裁剪或导入。
9. 通过火山录音识别做歌词时间轴对齐；失败时降级为粗略时间轴。
10. 打包剪映草稿；可选把分镜图转为 RunningHub 动态视频，再升级草稿。

## 5. 辅助工作台

| 模块 | 主要能力 |
| --- | --- |
| 对标监控 | 监控视频号账号、刷新作品、无水印下载、ASR 提取文案、AI 纠错与结构拆解、带文案创建任务 |
| 选品助手 | 抓取当当畅销榜、AI 过滤、可选豆瓣评分、导出 Excel、下载封面/内页、跳转对标或创建带货任务 |
| 人物素材库 | 按人物管理真图、拖拽排序、粘贴/导入，创作时作为分镜素材 |
| 草稿模板 | 管理和编辑剪映草稿布局 |
| 提示词模板 | 管理赛道化的改写、分镜、封面和绘图规则 |
| 画图实验室 | 独立测试提示词、风格和图片模型 |
| 配音实验室 | 独立测试音色、语速并导出 MP3，不写入任务历史 |
| 创作市场 | 购买、试用或上架提示词模板和画面风格；依赖原产品账户与积分后台 |
| 队列与历史 | 批量执行、状态跟踪、恢复、重命名、删除和批次摘要 |
