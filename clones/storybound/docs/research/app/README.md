# Storybound 桌面产品研究总览

研究对象：Storybound 1.13.1（公开教程标注更新日期 2026-07-14）。

结论：Storybound 不是“落地页 + 一个生成按钮”，而是一套本地优先的桌面视频生产工作台。当前版本包含三类主创作入口、三种图文执行模式、可断点编辑的任务流水线、批量队列、模板与素材资产，以及若干选品和对标辅助模块。

## 已确认的产品边界

- 三类主创作入口：图文任务、HTML 动画视频、音乐 MV。
- 图文任务三种执行模式：全自动、半自动、直接出片。
- 图文任务固定 7 步：预审、改写、分句、提示词、生图、配音、剪映草稿。
- 任务可以暂停、确认、修改中间产物、修复失败项、从任意可复用步骤续跑。
- 核心数据保存在本机 SQLite 和任务目录；模型调用、授权、积分同步等需要网络。
- 桌面端由 Tauri + React 构成，另带一个本地 sidecar 负责音频处理和剪映草稿打包。

## 研究原则

实现使用独立代码和用户自有的第三方 API 凭据，不复用原应用的授权机制或私有服务凭据。经用户明确要求，Storybound 1.13.1 客户端包中随前端分发的提示词模板已结构化提取，并按原赛道、画风和步骤映射接入本地流水线。
