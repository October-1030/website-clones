# Tracks and Pipeline Specification

## Overview
- Targets: `#tracks`, `#pipeline`
- Interaction model: track hover; pipeline static

## Computed styles
- Tracks desktop: 1152px wide, four 276px columns, 16px gap; cards are 276×150.55px, 12px radius, padding `24px 20px`.
- Tracks mobile: two 163.2px columns, 16px gap, total height 650.2px.
- Pipeline desktop: seven ~154.28px columns, 12px gap; each card is 118.3px high with `18px 14px` padding.
- Pipeline mobile: two 165.2px columns, 12px gap, total height 509.2px.

## Content
- Tracks: 人物故事, 健康图书, 民间故事, 文化科普, 绘本故事, 电商带货, 心灵鸡汤, 更多赛道.
- Pipeline: 文案预审, 智能改写, 影视分镜, 提示词生成, 批量生图, TTS 配音, 剪映打包.

## Responsive behavior
- Tracks: four columns desktop, two columns tablet/mobile.
- Pipeline: seven columns desktop, four at ≤980px, two at ≤600px.
